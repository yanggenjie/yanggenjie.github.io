我的软件清单

软件清单系列文章链接：

  • 我的软件清单 (Dec 22, 2022)

效率

  • everything：本地搜索最快的一个软件，想找什么文件，1s 就可以搜索到，几乎 0 延迟。比 windows 自带的搜索好。
  • utools：这个工具有很多插件，我用它来启动应用，作为取色器，翻译工具
  • thunderbird：开源免费的邮件工具

编程

  • virtualBox：开源免费，小巧但性能强劲的虚拟机。
  • vscode：宇宙最强”IDE”。
  • WindTerm：这个是 SSH 远程连接的终端工具，也是开源免费，而且还是绿色版的软件，对系统 0 入侵。
  • soutcetree：这是一个 git 版本控制器的图形界面工具，开源免费的。不用记住那么多 git 命令，基本的推送、分支新建、合并、拉取都可以应付，只有变基、冲突合并之类的 操作需要命令。

播放器

  • Potplayer：最强的本地视频播放器，我感受比较深的就是解码能力很强，本地播放视频的不二之选。
  • listen1：网络音乐播放器，聚合了各大平台的音乐，当然收费的还是没法听。开源免费
  • foobar2000：本地音乐播放器

浏览器

  • firefox：开发者首选
  • chrome：备用

笔记工具

typora：markdown 阅读、编器。

0.9.8 版本的都是免费的，1.0 以后改收费了 ，界面用不太习惯，就没购买了。这个只是用来做阅读器，编辑写作都是用 obsidian 或者 vscode 比较多。

obsidian：免费的一个笔记工具，markdown”文件管理器”

下载工具

NeatDownload：免费的下载工具，IDM 的替代。嗅探能力没 IDM 强，不过也差不了多少。

办公

  • freeplan：开源免费，思维导图工具
  • 稻壳阅读：免费的 epub、mobi 阅读工具。

翻译

  • copytranslation：开源免费，翻译软件。
  • goldenDict：本地词典软件，不过需要自己找到各种词典的库文件。
  • mdict：也是一个查词软件，和 goldenDict 差不多，goldenDict 强一点。

卸载清理

geek：免费的卸载工具，超轻量。

卸载后 geek 会扫描残留的注册表信息、文件夹残留，然后清理。

密码管理

keepassXC：开源免费密码管理器。

注册的账号越来越多，很多人喜欢多个账户用同一个密码，这是很危险的行为。

一旦你某个账号的密码泄漏了，攻击者就会使用相同的密码去尝试登录其它的网站，各类 app。行内称”撞库”。

所以使用密码管理器是有必要的，我注册的账号和密码，我自己都不知道，都记录在密码管理器里，每个账号都是随机的密码。

录制工具

  • captura：开源免费的录屏工具，这个要额外下载 ffmpeg 来配置，在线下载有点慢，可以自己先去下载 ffmpeg，然后手动配置 。
  • screenToGif：免费的 gif 录制、编器工具。
