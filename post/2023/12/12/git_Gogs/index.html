简介

之前有介绍过如何配置一个最简单的 git 服务器仓库，简单是简单，但操作过于繁琐，每次都要进入服务器执行命令，修改对应的配置，使用的时候不够简单，今天就介绍使用 gogs 搭建最简易的 git 服务器。

Windows 篇

操作步骤

  1. 去官网下载 gogs 二进制文件：https://gogs.io/docs/
  2. 解压出来，启动./gogs web默认 3000 端口。
  3. 然后访问http://localhost:3000根据提示操作就行，如果为了方便，可以选择 sqlite3，这样数据库环境都不用配置了。

操作演示

先去官网从文档里面找到二进制安装，选较新的版本，然后选 windows,下载 zip 压缩包

解压出来，进入解压的目录，右键打开 PowerShell 或者 cmd 窗口，输入./gogs.exe web：

回车之后，就会启动服务了

然后打开浏览器，访问http://localhost:3000

接下来填写这些配置就行了，这里为了方便体验，数据库方面选择 Sqlite3，然后其它可以保持默认，直接往下拉，点立即安装就行了，这些配置内容先使用默认的，后面如果想修改配置，就在 gogs.exe 同级目录下，创建custom/conf/app.ini这个路径的app.ini文件，在app.ini
文件里面修改这些配置，修改完之后重启服务就行了。

安装成功，接下来只需要注册一个账号，然后开始自己体验吧！
