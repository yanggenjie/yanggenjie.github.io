简介

程序员的代码仓库，总是需要一份托管在服务器上，这样既方便多台设备间的代码同步，也方便协作，留一份在服务器上，从数据备份的角度来说这是比较安全的。

一般我们都是使用别人搭建好的托管服务，比如 github、gitee、gitlab 等等这些托管平台。

自己搭建托管服务，主要是不想把代码公开到公网上，比如公司自己开发时写的代码，基本都是有保密协议的，代码不能随便泄露，放在别人的服务器上总归没有那么安全，最好的方式就是在自己的服务器上搭建一个托管服务，开源的 git 服务器框架也有很多，比如 Gitlab CE、Gitea、Gogs 等等，

其中安装最简单的就是 Gogs，但我这里是自己使用，要越简单越好，最好什么框架都不用，就用 git 原生的功能，还真有。
